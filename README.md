 Sistema de Cadastro e Lista de Tarefas

Este projeto é uma aplicação simples em HTML para cadastro e visualização de tarefas, dividida em duas páginas principais:

cadastro.html – Página para cadastrar novas tarefas

index.html – Página que exibe a lista de tarefas cadastradas

Ideal para estudos de front-end básico ou como base para um sistema maior.

 Funcionalidades

Adicionar novas tarefas com informações relevantes

Visualizar tarefas cadastradas

Interface simples e direta

Estrutura leve usando apenas HTML + JavaScript (opcional, caso exista no arquivo)

 Estrutura do Projeto
/
├── index.html                # Página de lista de tarefas
├── cadastro.html             # Página de cadastro de tarefas

 Como Usar

Baixe este repositório:

Abra o arquivo index.html no navegador para visualizar as tarefas.

Acesse cadastro.html para adicionar novas tarefas.
